Muamalat Application Platform (MAP)

The Muamalat Application Platform (MAP) is a comprehensive solution designed to manage and facilitate various Islamic financial transactions and activities. It includes both a web application and a mobile application, each tailored to provide users with a seamless, accessible, and secure experience.

  • Web Application

    • The web application serves as the core platform for managing, monitoring, and executing a wide range of financial activities based on Islamic principles (Muamalat). It is designed for both individual users and businesses, offering features that cater to different needs.

  • Mobile Application

    • The mobile application is a companion to the web application, offering users the ability to manage their Muamalat activities on the go. It is designed to be intuitive and user-friendly, ensuring that users can easily access all features from their smartphones.

ArRahnu Online Auction System (Pajak Gadai-i)

The ArRahnu Online Auction System, or Pajak Gadai-i, is a digital platform designed for the auctioning of collateralized items from Islamic pawnshops (Ar-Rahnu). This system ensures that auctions are conducted in accordance with Shariah principles, providing a transparent, secure, and user-friendly environment for both sellers (pawnshops) and buyers (bidders). The system is accessible through both web and mobile applications, catering to a wide audience and enabling convenient participation in the auction process.

  • Web Application

    • The web application serves as the primary platform for managing and participating in auctions of collateralized items. It is designed to facilitate all aspects of the auction process, from item listing to final sale, ensuring compliance with Islamic laws.

  • Mobile Application

    • The mobile application is designed to offer the same functionalities as the web application but optimized for smartphones and tablets. It allows users to participate in auctions, manage listings, and make payments from anywhere, providing flexibility and convenience.
